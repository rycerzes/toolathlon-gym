# Task Guide: Tech Course Video Resources

## Overview
Map Fireship YouTube videos to relevant university courses and create a curriculum resource map.

## Step 1: Find Courses
Query courses from the learning management system with names containing:
Analytics, Algorithms, Computing, Data, or Software

## Step 2: Match Videos
For each course, find 2-3 relevant Fireship videos by keyword matching:
- Analytics/Algorithms courses: match video titles containing "algorithm", "data", "code", "software"
- Computing courses: match video titles containing "code", "computing", "tech", "programming"
- Data courses: match video titles containing "data", "database", "code", "analytics"

## Deliverable 1: Curriculum_Video_Map.xlsx

Sheet: Course_Videos
Columns: Course_Name, Course_Code, Video_ID, Video_Title, View_Count, Duration_Min, Relevance_Match
Sort: Course_Name ASC, then View_Count DESC

Sheet: Summary (exactly 3 rows)
Row 1: Total_Courses_Mapped | [count]
Row 2: Total_Videos_Recommended | [count]
Row 3: Avg_Views_Recommended | [integer]

## Deliverable 2: Notion Page
Title: "Tech Course Video Resources"
Content: One section per course with recommended video titles
