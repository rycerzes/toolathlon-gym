Our accounting team needs a tax summary report generated from our online store order data. Please retrieve all orders that have a status of either completed or processing from our e-commerce platform. For these orders, compile a tax summary that breaks down the data by the billing state of each customer.

Create an Excel file called Tax_Summary_Report.xlsx in the workspace with two sheets. The first sheet should be named "By State" and should contain the following columns: State, Order Count, Total Sales, and Total Tax. List each billing state that appears in the qualifying orders, sorted by total tax in descending order so the highest tax states appear first. All monetary values should be rounded to two decimal places. The second sheet should be named "Overall" and should contain two columns called Metric and Value with the following four rows: Total Orders with the count of all qualifying orders, Total Sales with the sum of all order totals, Total Tax with the sum of all tax amounts, and Effective Tax Rate with the overall tax as a percentage of total sales rounded to two decimal places.

In addition to the Excel file, create a shared spreadsheet titled "Tax Summary for Accounting" so the accounting team can access the data online. This spreadsheet should have a single sheet named "Summary" containing the same Overall data with the Metric and Value columns and the same four rows as the Overall sheet in the Excel file.
