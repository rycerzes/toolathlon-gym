You are a support operations analyst tasked with benchmarking your company's customer support performance against industry standards. An external research firm has published customer satisfaction benchmarks that are available as a JSON API at http://localhost:30408/api/csat_benchmarks.json. This data includes industry average satisfaction scores and response time targets broken down by ticket priority level.

Start by fetching the CSAT benchmark data from the API. The data contains industry averages for High, Medium, and Low priority tickets including target satisfaction scores, acceptable response time thresholds, and resolution time benchmarks.

Next, query the support ticket database for your company's actual performance metrics. You need to retrieve the total ticket count, average customer satisfaction rating, average response time in hours, and the count of resolved versus unresolved tickets, all broken down by priority level (High, Medium, Low). Also retrieve performance data for each individual support agent showing their ticket count, average satisfaction rating, and average response time.

Write and run a Python script that performs an SLA compliance analysis. The script should compare your company's actual metrics against the industry benchmarks for each priority level. For each priority, calculate the satisfaction gap (your score minus benchmark), the response time gap (your time minus benchmark target), and determine an SLA compliance status of "Compliant" if both satisfaction and response time meet the benchmarks, "At Risk" if one metric is close to the threshold, or "Non-Compliant" if either metric significantly misses the target. Also identify the top 3 and bottom 3 agents by satisfaction rating.

Create a PowerPoint presentation called Support_Performance_Review.pptx in the workspace with at least 7 slides. The first slide should be a title slide with "Support Performance Benchmark Review" and the current date. The second slide should present the industry benchmark data from the API. The third slide should show your company's overall ticket metrics by priority. The fourth slide should compare company performance versus industry benchmarks with the gaps highlighted. The fifth slide should present the SLA compliance status for each priority level. The sixth slide should show agent performance rankings. The seventh slide should provide key takeaways and recommendations for improvement.

Send an email from analytics@support-team.example.com to managers@support-team.example.com with the subject "Support Performance Benchmark Analysis". The email body should summarize the key findings including the overall satisfaction score compared to the industry benchmark, which priority levels are SLA compliant, the names of the top performing agents, and any areas needing improvement.

Your workspace contains SLA_Policy_Reference.pdf with the company's internal SLA targets and reporting_template.json with formatting preferences.
