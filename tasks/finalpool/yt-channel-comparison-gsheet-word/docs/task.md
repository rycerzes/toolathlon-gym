You are a media analyst tasked with producing a comparative analysis of two prominent YouTube channels stored in your team's video database. The two channels are Fireship (a technology channel) and Veritasium (a science channel).

For each channel, compute the following metrics using the data in the video database: total number of videos, total view count across all videos, total like count across all videos, average view count per video rounded to the nearest whole number, average video duration in seconds rounded to the nearest whole number, average like rate calculated as total likes divided by total views multiplied by 100 rounded to 2 decimal places, and the title and view count of the single most-viewed video in that channel.

Create a cloud spreadsheet titled "Channel Comparison Analysis". This spreadsheet must contain two sheets.

The first sheet must be named Comparison. It should have exactly one header row followed by two data rows, one per channel. The columns must be in this order: Channel, Video_Count, Total_Views, Total_Likes, Avg_Views, Avg_Duration_Sec, Avg_Like_Rate, Top_Video_Title, Top_Video_Views.

The second sheet must be named Monthly_Fireship. It should show monthly publishing data for the Fireship channel only, with columns Month (formatted YYYY-MM) and Video_Count, sorted by Month ascending.

Next, create a Word document called Channel_Analysis_Report.docx in your workspace. The document must contain at least four sections with the following headings: Executive Summary, Channel Metrics Comparison, Key Findings, and Recommendations. The Executive Summary section should introduce the purpose of the report. The Channel Metrics Comparison section should include the computed statistics for both channels. The Key Findings section should highlight notable differences between the channels. The Recommendations section should offer strategic suggestions based on the data.

Finally, send an email to media@company.com with subject "YouTube Channel Comparison Report" and attach or reference the Channel_Analysis_Report.docx in your message body.
