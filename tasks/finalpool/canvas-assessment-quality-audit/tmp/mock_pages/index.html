Assessment Quality Standards Portal

These standards govern the quality of quiz and exam items across all university courses.

Item Analysis Thresholds

Metric Acceptable Range Description
Difficulty Index 0.3 - 0.8 Proportion of students answering correctly. Items outside this range are flagged for review.
Discrimination Index Minimum 0.2 Difference in performance between top and bottom groups. Low discrimination suggests item revision.
Reliability Target 0.7 Cronbach's alpha for overall assessment reliability.
Question Bank Minimum 15 Minimum number of questions per quiz for adequate content sampling.

For calculation methodology, see Assessment Methodology.
