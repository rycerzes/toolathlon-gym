The assessment office needs to conduct a quality audit of quiz assessments across all university courses. Start by visiting the assessment quality standards portal at http://localhost:30221 to retrieve the acceptable ranges for difficulty index, discrimination index, reliability targets, and minimum question counts. Also follow the link to the methodology page to understand how difficulty is calculated.

Read the exam policy document in the workspace for the university's official assessment guidelines, and review the assessment glossary for definitions of the psychometric terms used in the audit.

Then query the learning management system to gather data on every quiz across all courses. For each quiz, count the number of questions, calculate the average score across all submissions, and compute the difficulty index. The difficulty index is calculated as the average score divided by the total possible points, where total possible points equals points_possible multiplied by the question count. This produces a value between 0 and 1 representing the proportion of maximum points earned on average.

Create an Excel file called Assessment_Quality.xlsx with three sheets. The first sheet named "Quiz Overview" should contain columns for Course, Quiz, Question_Count, Avg_Score, and Avg_Difficulty (the difficulty index as a decimal between 0 and 1). Sort by course name then quiz name. The second sheet named "Flagged Items" should list only quizzes whose average difficulty index falls outside the acceptable range from the standards portal, with columns for Course, Quiz, Question_Count, Difficulty_Index, and Issue (either "Too Easy" if above the upper threshold or "Too Hard" if below the lower threshold). The third sheet named "Summary" should have rows for Total_Quizzes, Total_Questions (sum of all question counts), Flagged_Quizzes, and Flag_Rate (percentage of quizzes that were flagged).

Also create a Word document called Assessment_Report.docx containing a narrative quality audit report. Include an executive summary with the total number of quizzes analyzed and flags identified, a methodology section explaining the difficulty index calculation, a findings section discussing patterns in the flagged items, and a recommendations section for improving assessment quality.
