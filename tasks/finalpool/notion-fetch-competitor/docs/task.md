The product strategy team needs a competitive analysis for our SaaS project management tool. Competitor product data has been collected and saved as competitor_data.json in the workspace. The file contains information about our product (TaskFlow Pro) and four competitors.

Read the competitor data file to understand the competitive landscape. After reviewing the data, remember key competitive insights found during the analysis for future reference by saving notes to the memory file at memory/memory.json in the workspace.

Create an Excel file called Competitor_Analysis.xlsx in the workspace with three sheets. The first sheet should be named "Product Comparison" and should have the following columns: Company, Product_Name, Pricing_Tier, Monthly_Price, Key_Features (comma-separated list), Target_Market, Rating. Include one row per competitor for a total of 4 rows. For the Pricing_Tier column, use the middle tier for each competitor (pro for ProjectHub, standard for AgileBoard, growth for WorkStream, business for TeamSync). For Monthly_Price, use the price corresponding to that middle tier.

The second sheet should be named "Feature Matrix" and should have the following columns: Feature_Name, Our_Product, Competitor_A (ProjectHub), Competitor_B (AgileBoard), Competitor_C (WorkStream), Competitor_D (TeamSync). The values in each competitor column should be Yes, No, or Partial to indicate whether that product supports the feature. Include at least 6 features that are relevant across the products.

The third sheet should be named "SWOT Summary" and should have the following columns: Category, Description, Priority. The Category column should contain one of Strength, Weakness, Opportunity, or Threat. The Description column should contain a brief description of the item. The Priority column should be High, Medium, or Low. Include at least 4 rows covering all four SWOT categories (at least one row per category).

Finally, create a page in Notion titled "Competitive Analysis - Q1 2026" with a summary of the competitors, key differentiators between our product and the competition, pricing strategy recommendations, and action items for the product team.
