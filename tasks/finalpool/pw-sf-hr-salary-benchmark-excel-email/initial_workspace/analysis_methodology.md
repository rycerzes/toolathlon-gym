# Salary Benchmark Analysis Methodology

## Comparison Framework
Difference = Our_Avg_Salary - Industry_Benchmark
Difference_Pct = (Difference / Industry_Benchmark) * 100

## Status Classification
Above: Difference >= 0 (we pay at or above market)
Below: Difference < 0 (we pay below market)

## Overall Status
If more departments are Above than Below: "Competitive"
Otherwise: "Needs Attention"

## Salary Per Year Experience
Salary_Per_Year_Exp = Our_Avg_Salary / Avg_Experience
