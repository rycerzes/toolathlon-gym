The Dean wants a comprehensive learning analytics dashboard comparing course performance against institutional benchmarks. Start by visiting the analytics benchmark portal at http://localhost:30219 to retrieve the target values for key performance indicators. Also follow the link to the detailed benchmarks page for the performance tier definitions.

Then query the learning management system to calculate the following metrics for each of the 22 courses. Completion rate is the percentage of submissions that have been graded out of total submissions. Average grade is the mean current score across all enrolled students. Late submission rate is the percentage of submissions marked as late out of total submissions. Discussion participation count is the number of discussion topics in the course.

Review the analytics framework document in the workspace for the KPI calculation methodology, and the dashboard template file for the structure of the summary page.

Create an Excel file called Learning_Analytics.xlsx with three sheets. The first sheet named "Course KPIs" should contain columns for Course, Completion_Rate (percentage), Avg_Grade (numeric score), Late_Rate (percentage), Discussion_Count, and Meets_Benchmark (Yes only if completion rate is at least 75 percent AND average grade is at least 70 AND late rate is at most 15 percent). Sort alphabetically by course name. The second sheet named "Trends" should have columns for Course, Category (Completion, Grade, or Late Submissions), and Performance (Above/Below for completion and grade, Acceptable/High for late submissions). Each course should have three rows, one per category. The third sheet named "Summary" should have rows for Total_Courses, Meets_All_Benchmarks (count of courses meeting all targets), Below_Benchmark (count of courses missing at least one target), and Benchmark_Rate (percentage meeting all targets).

Finally, create a page in the notes system titled "Learning Analytics Dashboard" with sections summarizing the overall benchmark performance, listing courses that meet all targets, highlighting courses needing attention, and providing recommendations for improvement.
