Key performance indicators for course quality assessment.

Engagement Metrics Targets

KPI Target Description
Completion Rate 75% Percentage of submissions graded out of total submissions. Courses below this threshold need intervention.
Average Grade 70 Mean current score across enrolled students. Target ensures adequate learning outcomes.
Late Submission Threshold 15% Maximum acceptable percentage of late submissions. Courses above this need better deadline management.
Discussion Participation 3 topics minimum Minimum number of discussion topics per course to ensure student engagement.

A course meets the overall benchmark only if ALL individual KPI targets are satisfied.
