You are an academic coordinator at a university tasked with analyzing late submission patterns for Fall 2013 courses and creating calendar reminders for assignment deadlines.

First, identify all courses from the Fall 2013 semester. These courses have course codes ending in "2013J". There should be six courses: AAA-2013J, BBB-2013J, DDD-2013J, EEE-2013J, FFF-2013J, and GGG-2013J.

For each Fall 2013 course, analyze the submissions to determine how many were submitted late. A submission is considered late if its "late" field is true. Calculate the total number of submissions, the number of late submissions, and the late submission percentage for each course.

Create a Word document called "Late_Submission_Report.docx" in your workspace. The document should have the title "Fall 2013 Late Submission Analysis" at the top. Then include a section for each of the six Fall 2013 courses. Each section should include the course code, the course name, the total number of submissions, the number of late submissions, and the late submission percentage. After the per-course sections, include an overall summary with the total across all courses.

Next, create Google Calendar events for each assignment from the Fall 2013 courses that has a due date. Each event should have a summary in the format "[Course_Code]: [Assignment_Name] Due" (for example, "AAA-2013J: TMA 1752 Due"). Schedule each event on the assignment due date. Only create events for assignments that have a due date set.
