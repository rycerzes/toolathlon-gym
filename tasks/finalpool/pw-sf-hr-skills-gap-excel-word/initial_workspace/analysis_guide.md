# Skills Gap Analysis Guide

This guide provides the methodology for conducting the skills gap analysis.

## Salary Gap Calculation
Salary Gap = Department Average Salary - Industry Benchmark Salary

## Role to Department Mapping
Map industry roles to internal departments as follows:
Data Engineer, Software Developer -> Engineering
HR Manager -> HR
Sales Executive -> Sales
Financial Analyst -> Finance
Operations Manager -> Operations
Support Specialist -> Support

## Gap Percentage
Gap_Pct = (Our_Avg_Salary - Industry_Salary) / Industry_Salary * 100

## Summary Metrics
Departments_Below_Benchmark = count of departments where Salary_Gap < 0
Average_Salary_Gap = mean of all departments' Salary_Gap values
