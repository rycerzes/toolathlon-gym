Pull historical price data for all available stocks in the system. Exclude gold futures and the index from the main stock analysis, but use the Dow Jones index as the benchmark for comparison. Each stock represents its sector in this analysis. Use the terminal or your computation tools to calculate the momentum scores following the methodology in the PDF, 1-month, 3-month, and 6-month returns, then derive the composite momentum score with the specified weights. Generate the overweight, neutral, or underweight signals by comparing each stock's composite to the benchmark composite.

Create an Excel file called Sector_Rotation.xlsx in the workspace with three sheets. The first sheet should be named Momentum Analysis with columns Symbol, Company_Name, Sector, Latest_Price, Return_1M_Pct, Return_3M_Pct, Return_6M_Pct, Composite_Momentum, Benchmark_Momentum, and Signal. The second sheet should be named Relative Strength with columns Symbol, RS_1M, RS_3M, RS_6M, Avg_RS, and RS_Rank. Relative strength here means the excess return, which is the stock period return minus the benchmark period return for each timeframe. Avg_RS is the average of the three period excess returns. RS_Rank goes from 1 (strongest) to 5 (weakest) based on Avg_RS. Round RS values to 3 decimal places. Sort by Symbol.

The third sheet should be named Strategy Summary and contain label-value pairs in two columns. Include Analysis_Date (the latest data date), Benchmark_Return_6M, Overweight_Count, Neutral_Count, Underweight_Count, Top_Momentum_Stock (symbol with highest composite), Bottom_Momentum_Stock (symbol with lowest composite), Avg_Composite_Momentum (average of all 5 stocks' composites rounded to 2 decimals), and Portfolio_Signal which should be Bullish if more than half the stocks are Overweight, Bearish if more than half are Underweight, otherwise Mixed.

After creating the Excel file, set up a Notion database called Sector Research with the following properties: Stock as title, Sector as a select property, Signal as a select property with options Overweight Neutral and Underweight, Momentum as a number property, and Last_Updated as a date property. Add an entry for each of the 5 stocks with their respective sector, signal, composite momentum score, and today's date.
