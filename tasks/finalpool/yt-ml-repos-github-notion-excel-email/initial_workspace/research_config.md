Research Cataloging Guide

Purpose
This guide explains how to catalog ML research resources from the ML Tech Reviews channel.
The channel has a curated playlist of 7 videos, each covering a specific open-source ML project.

Metadata to Capture for Videos
For each video, record: the unique video identifier, the title, the primary ML topic or technique,
the repository URL found in the video description, the publication date, and the view count.

Metadata to Capture for Papers
For each paper related to a video topic, record: the paper identifier, title, authors
(formatted as a comma-separated string), publication date, the topic it belongs to,
and the video identifier it is related to.

Linking Videos to Papers
Match papers to videos by topic. Use the topic_mapping.json file for the list of expected topics.
Each topic should have at least one video and ideally one or more related papers.

Knowledge Base Structure
Create a single hub page with an introduction, then a database of all items (videos and papers)
with properties for Name, Type (Video or Paper), Topic, URL, and Status.

Summary Sheet
The Summary sheet should aggregate per topic: how many videos, how many papers,
and how many distinct repository URLs were found.
