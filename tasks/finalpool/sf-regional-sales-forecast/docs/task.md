The finance team needs a regional sales forecast for Q1 2026 that combines historical sales performance with external economic indicators. The economic indicator data is available as a JSON file at http://localhost:30212/api/economic_indicators.json which provides GDP growth rates, consumer confidence indices, and inflation rates for each of our five sales regions. You will need to fetch this data and combine it with historical sales figures from the data warehouse to produce a forecast.

Start by reading the forecast methodology document in the workspace which describes the specific formula to use, and the assumptions file which lists the key assumptions underlying the forecast. Then fetch the economic indicators from the API endpoint and query the data warehouse for monthly sales revenue by region for the last three months of 2025, specifically October, November, and December.

The forecast methodology is as follows. For each region, calculate the 3-month trailing average of monthly revenue using October through December 2025 data. Then apply an economic adjustment using the formula: Adjusted Monthly Forecast equals the Trailing 3-Month Average multiplied by the quantity one plus GDP Growth Rate divided by 100. The quarterly forecast for each region is then three times the adjusted monthly forecast.

Create an Excel file called Sales_Forecast.xlsx in the workspace with three sheets. The first sheet named Historical should have columns Region, Month in YYYY-MM format, Orders as integer count, and Revenue rounded to two decimal places, with one row for each region-month combination for October through December 2025, sorted by region then month. The second sheet named Forecast should have columns Region, Trailing_3M_Avg rounded to two decimals, GDP_Growth_Pct, Economic_Adjustment factor rounded to four decimals, Adjusted_Monthly_Forecast rounded to two decimals, and Next_Quarter_Forecast rounded to two decimals, with one row per region sorted alphabetically. The third sheet named Summary should have Metric and Value columns listing Total_Q4_2025_Revenue as the sum of all Oct through Dec revenue, Total_Monthly_Average, Total_Q1_2026_Forecast as the sum of all regional quarterly forecasts, QoQ_Growth_Pct showing the percentage change from Q4 actual to Q1 forecast rounded to one decimal, Highest_Forecast_Region, Highest_Forecast_Amount, Lowest_Forecast_Region, Lowest_Forecast_Amount, Highest_GDP_Growth_Region, and Regions_Count.

Create a Word document called Forecast_Report.docx in the workspace with sections for executive summary describing the overall forecast outlook, methodology explaining the formula and data sources used, regional analysis discussing each region's forecast with specific numbers, key assumptions listing the major assumptions, and recommendations suggesting where to focus sales efforts based on the forecast and economic outlook.
