# Analysis Guide - Arxiv Litreview

## Methodology
This analysis compares internal metrics against external benchmarks.
Gap = Internal Value - External Benchmark
Positive gap indicates we exceed the benchmark.
Negative gap indicates we fall short.

## Priority Actions
Focus on areas with the largest negative gaps first.
Items within 5% of benchmark are considered "On Track".

## Reporting
Include data sources, methodology, and confidence levels in all reports.
