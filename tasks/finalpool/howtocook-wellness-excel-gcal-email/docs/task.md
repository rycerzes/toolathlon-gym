You are a corporate wellness coordinator. Your task is to create a 5-day healthy meal plan for the workweek, schedule daily wellness lunch breaks in the company calendar, and communicate the program to all staff.

Your task has three parts.

Part 1: Search the recipe database to build a 5-day meal plan covering Monday through Friday. For each day, select three recipes: one for breakfast, one for lunch, and one for dinner. Choose recipes that are varied, nutritious, and suitable for a corporate wellness program. For each recipe, note the recipe name, category, and any relevant health notes.

Part 2: Create an Excel file called Wellness_Meal_Plan.xlsx in the workspace with two sheets. The first sheet must be named "Meal Plan" with the following columns in this exact order: Day, Meal_Type, Recipe_Name, Category, Notes. Add one row per meal (breakfast, lunch, dinner) for each of the 5 days, for a minimum of 15 rows. The days should be Monday, Tuesday, Wednesday, Thursday, and Friday. The Meal_Type values should be Breakfast, Lunch, or Dinner. The Notes column should include a brief nutritional benefit or wellness note for each recipe. The second sheet must be named "Summary" with two columns: Metric and Value. Include these rows: Total_Recipes (minimum 15), Unique_Categories (the number of distinct recipe categories used), Days_Covered (must be 5).

Part 3: Schedule 5 daily "Healthy Lunch Break" events in Google Calendar, one for each weekday (Monday through Friday) of the current task week. Each event should last 1 hour and start at 12:00 PM noon. Use the task launch time to determine the correct week. The event title must be exactly "Healthy Lunch Break".

After completing the above, send an email to all-staff@company.example.com from wellness@company.example.com with the subject "5-Day Corporate Wellness Meal Plan". The email body must include a brief description of the wellness program goals (reducing stress and improving focus), a summary of the meal plan (mentioning 5 days and 15 meals), and encouragement for staff to participate in the daily Healthy Lunch Break calendar events.

The initial workspace contains Wellness_Program_Guide.md describing the corporate wellness context and goals.
