Our support operations team needs a comprehensive performance review based on our support ticket data warehouse. Please query the support center database to gather ticket statistics broken down by priority level and by issue type. For each priority level, calculate the total number of tickets, the average response time in hours rounded to two decimal places, and the average customer satisfaction score rounded to two decimal places. You should also check what percentage of tickets met their SLA response time targets by comparing each ticket's response time against the SLA policy response target hours for that priority level. Additionally, retrieve the complete list of support agents with their team assignments and skill levels.

Once you have all the data, create an Excel file named Support_Analysis.xlsx in the workspace. The file should contain four sheets. The first sheet should be named "Agent Roster" and should list all agents sorted by name with columns Agent_Name, Team, and Skill_Level. The second sheet should be named "Priority Analysis" with columns Priority, Ticket_Count, Avg_Response_Hours, Avg_Satisfaction, and SLA_Compliance_Pct where the compliance percentage represents the proportion of tickets whose response time was within the SLA target for that priority. The third sheet should be named "Issue Type Analysis" with columns Issue_Type, Ticket_Count, Avg_Response_Hours, and Avg_Satisfaction, sorted by ticket count in descending order. The fourth sheet should be named "Summary" with two columns Metric and Value containing the following rows: Total_Agents with the count of agents, Total_Tickets with the overall ticket count, Overall_Avg_Response_Hours with the overall average response time rounded to two decimal places, and Overall_Avg_Satisfaction with the overall average satisfaction rounded to two decimal places.

After creating the Excel file, also create a shared spreadsheet containing just the summary data so that other stakeholders can view it. The spreadsheet should be titled "Support Performance Summary" and should have a single sheet called "Summary" with the same Metric and Value columns and the same four rows as the Summary sheet in the Excel file.
