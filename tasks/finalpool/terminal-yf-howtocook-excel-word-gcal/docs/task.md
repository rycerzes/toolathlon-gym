You are a corporate wellness coordinator planning a nutrition-aware meal program for Q2 2026. Start by reading the wellness_policy.md file in your workspace which outlines the program requirements, nutritional guidelines, and target offices. Also read budget_constraints.csv which contains the monthly budget allocation and employee headcount for each office location, and nutrition_standards.json which defines daily calorie and protein targets.

To estimate food costs, you need to account for commodity price trends. Look up recent stock price data for XOM (ExxonMobil, as a proxy for energy and transportation costs affecting food prices) and GC=F (gold futures, as an inflation hedge indicator). Pull the closing prices for the most recent 5 trading days available for each symbol. Write a Python script called compute_cost_index.py that reads these prices and computes a "food cost index" as follows: take the average closing price of GC=F over those 5 days, divide by 5000, then multiply by 1.0 plus the percentage change between the earliest and latest XOM closing prices in the 5-day window divided by 100. The resulting index represents a cost multiplier. Run the script and save the output to cost_index.json containing the fields gold_avg, xom_change_pct, and cost_index.

Next, query the recipe database for recipes in three categories that map to meal types: use the breakfast category for breakfast options, the vegetable dish category for lunch options, and the meat dish category for dinner options. Write a Python script called plan_meals.py that takes the recipe lists and the cost index, then for each category selects the 3 recipes with the lowest difficulty rating. If there are ties in difficulty, pick alphabetically by name. For each selected recipe, estimate a per-serving cost by multiplying the number of ingredients by 2.5 by the cost index value. Run the script and save output to recommended_meals.json with arrays for breakfast, lunch, and dinner, each containing objects with name, difficulty, ingredient_count, and estimated_cost fields.

Check the shared calendar for any existing events during the weeks of March 16 to April 10, 2026. You need to schedule 4 "Wellness Wednesday" workshop sessions on Wednesdays during that window, but you must avoid any Wednesday that already has an event. If a Wednesday conflicts, move that session to the next available Thursday instead. Each workshop runs from 12:00 to 13:00. The four session topics in order are: "Healthy Breakfast Habits", "Plant-Based Lunch Ideas", "Balanced Dinner Planning", and "Meal Prep Workshop". Create these 4 events on the calendar with these exact summaries.

Create an Excel workbook called Wellness_Program_Plan.xlsx with four sheets. The first sheet "Cost_Analysis" should have columns Month, Gold_Avg, XOM_Change_Pct, and Cost_Index, with one row for "March 2026" using the values from cost_index.json, all numbers rounded to 2 decimal places. The second sheet "Meal_Plan" should have columns Week (1 through 4), Breakfast, Lunch, Dinner, Daily_Cost, and Nutrition_Score. Assign the 3 breakfast recipes across weeks 1-3, the 3 lunch recipes across weeks 1-3, and the 3 dinner recipes across weeks 1-3. For week 4, repeat the lowest-cost option from each meal type. Daily_Cost is the sum of the three meal estimated costs for that week. Nutrition_Score is calculated as 100 minus the sum of difficulties of the three meals multiplied by 5. The third sheet "Event_Schedule" should have columns Date, Topic, Time, and Duration_Minutes, listing the 4 scheduled wellness events sorted by date, with Time as "12:00" and Duration_Minutes as 60. The fourth sheet "Budget_Summary" should have columns Office, Headcount, Monthly_Budget, Meal_Program_Cost, and Variance. Meal_Program_Cost for each office equals headcount times the average Daily_Cost across all 4 weeks times 22 working days. Variance is Monthly_Budget minus Meal_Program_Cost. Include a final row labeled "Total" that sums Headcount, Monthly_Budget, Meal_Program_Cost, and Variance.

Create a Word document called Wellness_Plan_Report.docx. It should have a title "Q2 2026 Corporate Wellness Plan", followed by sections for Program Overview summarizing the wellness policy goals, Cost Analysis discussing the commodity-based cost index methodology and results, Meal Recommendations listing all 9 selected recipes with their estimated costs, Event Calendar describing the 4 scheduled workshops with dates, and Budget Summary showing per-office budget allocation and overall variance.

Finally, write a Python script called validate_budget.py that reads the budget_constraints.csv and cost_index.json, computes the total meal program cost across all offices using the same formula as above, and prints whether the total program cost is within the total budget. Run this script.
