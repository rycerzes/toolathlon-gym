Module Review Template

Creative Computing and Culture - Fall 2014

Purpose:
This template guides the teaching team through weekly module review sessions.
Each Saturday meeting should cover the following agenda items.

Agenda:
1. Review of module completion status (15 minutes)
2. Discussion of student progress and common difficulties (20 minutes)
3. Action items for upcoming week (15 minutes)
4. Planning for next module content (10 minutes)

Module Completion Checklist:
- All module items accessible to students
- Assignment rubrics clearly published
- Discussion prompts updated
- Video resources working correctly
- Quiz questions reviewed for clarity

Status Definitions:
Not Started - Module has not yet been opened to students
In Progress - Module is currently active and students are working through it
Completed - Module has been finished and grades submitted

Notes:
Please update the module tracker in the team workspace after each review meeting.
Document any issues or feedback from students for instructor review.
