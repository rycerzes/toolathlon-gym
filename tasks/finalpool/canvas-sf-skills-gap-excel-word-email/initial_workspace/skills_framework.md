# Skills Gap Analysis Framework

Assessment Model:
1. Map each department's core skill requirements
2. Evaluate current training offerings coverage
3. Cross-reference employee performance with training
4. Calculate gap score (inverse of coverage * performance)

Gap Score Calculation:
- 1-3: Well covered, minimal gaps
- 4-6: Moderate gaps, improvement needed
- 7-10: Critical gaps, immediate action required

Priority Classification:
- Critical: Performance below average + low training coverage
- Important: Either below-average performance or low coverage
- Nice-to-have: Above average but room for improvement
