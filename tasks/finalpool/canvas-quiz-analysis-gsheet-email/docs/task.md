The academic coordinator wants to track quiz performance trends across courses to identify students who need additional support. Analyze quiz submission data from all courses to compute average scores per quiz per course.

Create a Google Sheet called "Quiz Performance Tracker" with one sheet named "Quiz Scores" containing the following columns: Course_Name, Quiz_Title, Total_Attempts, Avg_Score (rounded to 1 decimal), Max_Score, and Pass_Rate (the percentage of attempts where the score is at least 60 percent of the maximum possible score, rounded to 1 decimal). Only include quizzes that have at least one submission with a non-null score. Sort the rows by Course_Name, then Quiz_Title.

Also create a Word document called Quiz_Performance_Summary.docx that contains a summary section describing the overall quiz performance trends, a table showing the top performing quizzes by average score, and a recommendations section suggesting which courses need additional student support based on pass rates.

Email the findings to academic_coordinator@university.edu with subject "Quiz Performance Analysis" and include a brief summary of the key findings in the email body.
