I need a report on all refunded orders from our online store. Please look up every order that has a status of "refunded" and gather the relevant details for each one including the order ID, the customer name from the billing information, the refund total amount, and the date the order was created.

Create an Excel file called Refund_Report.xlsx in the workspace. The first sheet should be named "Refund Details" with columns Order_ID, Customer_Name (the billing first and last name combined), Total (the refund amount rounded to 2 decimal places), and Date (formatted as YYYY-MM-DD). Sort the rows by Order_ID in ascending order.

The second sheet should be called "Summary" with two columns Metric and Value containing three rows: Total_Refunds (the count of refunded orders), Total_Amount (the sum of all refund totals rounded to 2 decimal places), and Avg_Refund (the average refund amount rounded to 2 decimal places).

Once the report is ready, send a summary email to manager@store.example.com from analytics@store.example.com with the subject "Refund Analysis Report". The body should state the total number of refunded orders, the total refund amount, and the average refund value.
