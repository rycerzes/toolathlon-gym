Room Booking Rules and Time Preferences

Review sessions should be scheduled during the following preferred time slots. Weekday sessions run from 4pm to 6pm. Saturday sessions run from 10am to 12pm and should only be used when weekday slots are unavailable.

All rooms must be booked at least 48 hours in advance. Each session should last 90 minutes to allow adequate coverage of the material. No more than one session per course per day.

When scheduling multiple courses, distribute sessions across the week to avoid conflicts for students enrolled in multiple courses that need review.
