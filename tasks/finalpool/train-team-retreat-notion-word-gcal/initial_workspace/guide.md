# Team Retreat Planning Guide: 5-Person Trip to Qufu

## Overview
This guide defines the output requirements for planning a 5-person team retreat to Qufu, covering the Word document, calendar events, and Notion page structure.

## Word Document Sections

### Section 1: Retreat Overview
- Purpose of the retreat
- Dates (outbound and return)
- Number of participants and names
- Total travel budget

### Section 2: Outbound Journey
Table with columns: Participant_Name, Train_Number, Departure_Station, Departure_Time, Arrival_Time, Class, Price_CNY
Include a paragraph noting the selected train and why it was chosen (e.g., best balance of departure time and cost).

### Section 3: Return Journey
Same table structure as Outbound Journey for the return trip.

### Section 4: Schedule Notes
- Arrival time in Qufu and suggested check-in
- Any team activities or coordination notes
- Departure reminder time (suggested buffer before train)

## Calendar Event Format
Create two calendar events shared with all 5 participants:

1. Outbound event
   - Title: "Team Retreat - Outbound Journey to Qufu ([Train Number])"
   - Start: Departure datetime
   - End: Arrival datetime
   - Description: List all participant names and ticket class

2. Return event
   - Title: "Team Retreat - Return Journey to Beijing ([Train Number])"
   - Start: Departure datetime
   - End: Arrival datetime
   - Description: List all participant names and reminder to check in early

## Notion Page Structure
Create a Notion page titled "Team Retreat Qufu [Year]" with the following sections:

1. Overview (dates, headcount, purpose)
2. Travel Details (outbound and return train info as a table)
3. Participant List (names and any special notes)
4. Budget Summary (total cost, cost per person)
5. Action Items (checklist: book tickets, confirm hotel, send calendar invites)
