Our finance team needs to plan the travel budget for sending eight employees from Beijing to Shanghai for a business conference on March 10, 2026. Please look up all available high-speed trains on the Beijing to Shanghai route for that date and retrieve their seat class options with prices.

Create an Excel workbook called Travel_Budget.xlsx in the workspace. Add a sheet called Seat_Options that lists all seat classes across all available Beijing-to-Shanghai trains on March 10, 2026, with the following columns: Train_Code, Seat_Class, Price_Per_Ticket_CNY, Available_Seats. Each train and seat class combination should have its own row.

Add a second sheet called Budget_Scenarios with three scenarios for sending eight people. Each scenario row should have columns: Scenario, Train_Code, Seat_Class, Price_Per_Person, Total_Cost_8_People. Define three scenarios: first, the Budget scenario using second class on the cheapest available Beijing-Shanghai train (find the train code and price from the seat options you retrieved); second, the Standard scenario using first class on a recommended mid-morning train (find the appropriate train code and price); third, the Premium scenario using business class on that same recommended train (find the price from your data). Calculate Total_Cost_8_People as Price_Per_Person multiplied by 8 for each row.

Add a third sheet called Summary with columns Label and Value. Include these rows in order: Cheapest_Scenario (the scenario name with the lowest total cost), Most_Expensive_Scenario (the scenario name with the highest total cost), Price_Difference_CNY (the difference between the most expensive and cheapest total cost), Budget_Total_8_People (the Budget scenario total), Standard_Total_8_People (the Standard scenario total), Premium_Total_8_People (the Premium scenario total).

Create a survey form titled "Business Trip Travel Preference Survey" with exactly three questions. The first question should be a multiple choice question asking "Which train departure time do you prefer?" with options listing each available Beijing-Shanghai train's departure time and code, for example "07:00 - G11" and "09:00 - G1". The second question should be a multiple choice question asking "What is your preferred seat class?" with options: Second Class, First Class, and Business Class. The third question should be a text question asking "Any special travel requirements or comments?".

Send an email to finance@company.com with the subject "Travel Budget Analysis for Beijing-Shanghai Conference". The email body should include the three scenario totals for 8 people: Budget total, Standard total, and Premium total, and the price difference between the most and least expensive scenarios.
