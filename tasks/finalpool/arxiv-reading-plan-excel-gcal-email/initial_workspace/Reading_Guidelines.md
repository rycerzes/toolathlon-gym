LLM Agent Research Reading Group - Guidelines
=============================================

Format:
- Weekly sessions, one paper per session
- Sessions are 90 minutes long (10:00 - 11:30 UTC every Monday)
- Each session is led by one rotating presenter

Discussion Structure:
1. Paper summary by presenter (15 minutes)
2. Group discussion of key contributions (30 minutes)
3. Critical analysis and limitations (20 minutes)
4. Connections to current work (15 minutes)
5. Wrap-up and next session preview (10 minutes)

Preparation:
- All members should read the paper before the session
- Prepare at least 2 discussion questions
- Note any connections to your current research

Paper Assignment:
- Papers are assigned in the order listed in paper_ids.txt
- Session 1 starts on Monday March 9, 2026
- Subsequent sessions are each Monday thereafter (weekly)
- 8 papers total = 8 weeks of sessions

Topics Covered:
This reading plan focuses on LLM agent capabilities including:
- Tool use and API integration
- Multi-step reasoning
- Benchmark evaluation
- Retrieval-augmented generation
- Chain-of-thought and tree-of-thought prompting
