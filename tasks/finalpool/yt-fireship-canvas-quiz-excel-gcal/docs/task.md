You are a programming instructor who uses short online tutorial videos as supplementary material for your web development course. Your task is to find videos about TypeScript from a popular tech education channel, use those videos as the basis for a new course quiz, build a grade-tracking spreadsheet, schedule quiz-related events on the shared calendar, and notify students by email.

Begin by searching the video database for content related to TypeScript. Retrieve the top five TypeScript-related videos ranked by view count, and for each video retrieve the full details including title, description, duration, and any associated tags. From each video's title and description, identify three key programming concepts that the video covers.

Next, look up the available courses in the learning management system to find the web development or programming course where this quiz will be added. Review the existing assignments in that course to understand what has already been assigned. Then create a new quiz in that course with the title "TypeScript Essentials Quiz", a time limit of 30 minutes, and a total point value of 50 points. Add ten multiple-choice quiz questions to this quiz, creating two questions per video based on the concepts identified from each video's description. Each question should have four answer choices and one correct answer.

Create a spreadsheet file named TypeScript_Course_Tracker.xlsx with three sheets. The first sheet should be named Videos and contain one row per video with columns Video_ID (the video identifier), Title (full video title), Topic (the primary programming concept), Duration_Min (video duration in minutes), Concepts_Count (number of key concepts extracted, which should be 3 for each), and Quiz_Question_Count (number of quiz questions generated from this video, which should be 2 for each). This sheet must have at least four data rows. The second sheet should be named Quiz_Questions and contain one row per quiz question with columns Question_No (1 through 10), Video_Source (the video ID the question is based on), Concept (the concept being tested), Question_Text (the full question text), Correct_Answer (the correct option text), and Points (point value, which should be 5 for each question). This sheet must have at least eight data rows. The third sheet should be named Grade_Template with columns Student_ID, Student_Name, Score, Percentage, and Grade. This sheet should have only the header row since students have not yet taken the quiz.

Check the shared calendar for any events already scheduled on or around March 25 and March 26, 2026. Create a calendar event on March 25, 2026 titled "TypeScript Quiz Deadline" with an all-day description noting that the quiz closes at 23:59. Create a second calendar event on March 26, 2026 from 14:00 to 15:00 titled "TypeScript Quiz Review Session" for the in-class review of quiz answers.

Send an email to students@webdev.edu with subject "TypeScript Essentials Quiz Now Available". The email body should announce that the quiz has been posted in the course, state the deadline of March 25 at 23:59, mention that a review session is scheduled for March 26 from 14:00 to 15:00, and encourage students to review the linked TypeScript tutorial videos before attempting the quiz.
