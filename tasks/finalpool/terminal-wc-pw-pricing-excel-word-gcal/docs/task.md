The pricing team needs to conduct a competitive analysis by comparing our online store product prices against a competitor's publicly available catalog. There is a PDF file called Pricing_Strategy_Guide.pdf in the workspace that describes the company's pricing philosophy and margin targets. There is also a JSON file called margin_targets.json with minimum margin percentages by category. Please review both files first.

Start by pulling our complete product catalog from the e-commerce platform, including each product's name, regular price, and category information. You should have products across several categories like Electronics, Headphones, Speakers, and others.

Next, visit the competitor pricing page at http://localhost:30417/pricing.html using a browser. This page contains a table of competitor products organized by category with product names and their prices. Extract all the competitor product data from that page.

Using the command line, write and run a Python script that performs the price comparison analysis. The script should match products by category, calculate average prices per category for both our products and competitor products, compute the price gap percentage for each category (calculated as (our_avg minus competitor_avg) divided by competitor_avg times 100, rounded to 1 decimal place), and generate pricing recommendations. For each category, if our average price is more than 10 percent above the competitor, recommend "Consider price reduction". If our price is more than 10 percent below, recommend "Maintain competitive advantage". Otherwise recommend "Monitor pricing". Save the script output to a file called price_analysis_output.txt in the workspace.

Create an Excel file called Competitive_Pricing_Report.xlsx in the workspace with three sheets. The first sheet should be called "Our_Products" with columns Product_Name, Regular_Price, and Category listing all products from our store sorted by category then by name. The second sheet should be called "Competitor_Prices" with columns Product_Name, Price, and Category listing all products scraped from the competitor page sorted by category then by name. The third sheet should be called "Price_Gap_Analysis" with columns Category, Our_Avg_Price, Competitor_Avg_Price, Gap_Pct, and Recommendation. Sort by category name alphabetically. Only include categories that appear in both our catalog and the competitor catalog.

Create a Word document called Pricing_Strategy_Report.docx in the workspace. The document should have a title "Competitive Pricing Analysis Report", an executive summary paragraph describing the overall pricing position, a section for each overlapping category with the price comparison details and recommendation, and a conclusion with strategic next steps.

Finally, schedule price review meetings in the calendar. Create one meeting for each category that appears in the Price_Gap_Analysis sheet. Each meeting should have a summary like "Price Review: [Category Name]" and be scheduled on different days starting from March 10, 2026, each lasting one hour starting at 2:00 PM. The description should mention the gap percentage for that category.
