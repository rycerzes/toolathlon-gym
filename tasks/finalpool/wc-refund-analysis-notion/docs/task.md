You are a finance analyst responsible for preparing a monthly refund analysis report for the company.

Your goal is to analyze all refund data from the store, organize findings in a Notion workspace, create a detailed Excel report, and email the results to the CFO.

First, look up all refunds from the store. For each refund, collect the refund ID, order ID, refund amount, reason for refund, and the date the refund was created.

Create an Excel file called "Refund_Analysis.xlsx" in your workspace with two sheets:

The first sheet should be named "Refund Details" and contain columns: Refund_ID, Order_ID, Amount, Reason, Date. Include every refund found, sorted by refund ID.

The second sheet should be named "Summary" with the following rows of information: Total_Refunds (the total number of refunds), Total_Refund_Amount (the sum of all refund amounts), Avg_Refund_Amount (the average refund amount), Refund_Rate (calculated as the number of refunds divided by the total number of orders, expressed as a percentage).

Next, create a Notion page titled "Refund Analysis Dashboard". On this page, create a database called "Refund Trends" that contains summary entries about the refund patterns, including total count, total amount, and the most common refund reason.

Finally, send an email from finance@store.com to cfo@company.com with the subject "Monthly Refund Analysis Report". The body of the email should include a summary of the key findings: total number of refunds, total refund amount, average refund amount, and the most common refund reason.
