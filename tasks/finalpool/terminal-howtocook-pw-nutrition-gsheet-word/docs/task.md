You are a nutritionist creating a healthy meal plan for a corporate wellness program. You need to gather recipes, compare them against recommended daily nutritional standards, and produce a comprehensive diet plan document.

Start by reading the Wellness_Program_Brief.pdf in your workspace which describes the corporate wellness initiative goals and dietary requirements.

Search the recipe database for breakfast recipes. Get at least three breakfast recipe options with their full details including ingredients and preparation steps. Then search for lunch recipes and get at least three options. Finally search for dinner recipes and get at least three options. You should have at least nine recipes total across the three meal categories.

Visit the nutrition standards reference page at http://localhost:30404/nutrition_standards.html. This page contains a table of daily recommended values for key nutrients including calories, protein, carbohydrates, fat, fiber, and sodium. Extract all the recommended daily values from this page.

Write and run a Python script called nutrition_calculator.py in your workspace. The script should define the nutritional estimates for each recipe you gathered (you can use reasonable estimates based on the recipe ingredients: for example a rice-based dish might have approximately 400 calories, 12g protein, 65g carbs, 8g fat, 3g fiber, and 800mg sodium). For each meal slot (breakfast, lunch, dinner), pick the recipe with the most balanced nutrition profile. The script should calculate the total daily nutrition if one recipe from each meal is consumed, and compare it against the recommended daily values from the standards page. Output the percentage of daily recommended value met for each nutrient.

Create a cloud spreadsheet called "Nutrition Dashboard". Add a sheet called "Recipe Comparison" with columns Recipe_Name, Meal_Type, Calories, Protein_g, Carbs_g, Fat_g, Fiber_g, and Sodium_mg. Include one row for each of the nine or more recipes you gathered. Add a second sheet called "Daily Plan" with columns Meal, Selected_Recipe, Calories, Protein_g, Carbs_g, Fat_g, Fiber_g, Sodium_mg, and Pct_Daily_Calories. Include three rows for breakfast, lunch, and dinner with the selected recipes, plus a totals row.

Create a Word document called Wellness_Diet_Plan.docx in your workspace. The document should have a title "Corporate Wellness Diet Plan". Include a section on the recommended daily nutritional standards with the values from the reference page. Include a section listing all recipes considered, organized by meal type. Add a section with the recommended daily meal plan showing the selected breakfast, lunch, and dinner recipes along with their nutritional profiles. End with a section analyzing how well the proposed plan meets the daily recommended values and any nutritional gaps that should be supplemented.
