Add a required scale question asking How many home-cooked meals do you have per week, where the scale goes from 1 to 7. Add an optional text question asking Are there any specific ingredients you want to avoid.

Then browse through the recipe collection and select at least six recipes representing at least three different categories. Create a Notion page titled Healthy Recipe Knowledge Base. The page should contain blocks for each selected recipe that list the recipe name, the category it belongs to, and a brief description of the recipe.

The first sheet should be called Recipes and contain columns for Recipe_Name, Category, and Description, with one row for each of the selected recipes. The second sheet should be called Category Summary and show each category name and how many recipes you found in it.
