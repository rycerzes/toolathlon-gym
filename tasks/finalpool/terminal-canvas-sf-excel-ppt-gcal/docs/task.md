You are a workforce development analyst tasked with bridging the gap between university curriculum and industry workforce needs. Your goal is to compare what is being taught in courses with the actual skill and education profiles found in the corporate workforce, identify gaps, and present recommendations to an advisory board.

Begin by reading the Skills_Framework.pdf in your workspace, which outlines industry skill categories and the benchmarks used to evaluate curriculum alignment. Also read the gap_analysis_config.json file, which defines the mapping rules between academic course structures and workforce competency areas, along with threshold values for flagging significant gaps.

From the learning management system, pull detailed information about four courses. Specifically, retrieve data for the courses with identifiers 1 through 4. For each course, you need the course name, total student enrollment, the number of assignments, and the number of quizzes. These courses cover topics in applied analytics and biochemistry across two academic years.

From the enterprise data warehouse, query the human resources database to obtain a workforce profile. For each department, retrieve the headcount, the average salary, the average performance rating, and the average years of experience. Also pull the education level distribution across the entire workforce, showing how many employees hold each type of degree or credential.

Using command-line tools, write and execute a Python script called gap_analyzer.py in your workspace. The script should compute a gap analysis by mapping course assessment intensity to workforce competency expectations. For each of the four skill areas defined in the framework document, calculate a curriculum score based on the number of assessments in the relevant courses, normalized to a ten-point scale. The industry demand score should be derived from the workforce performance and education data. The gap score is the absolute difference between curriculum score and industry demand. A gap score above 3.0 should be flagged as Critical priority, between 1.5 and 3.0 as High priority, and below 1.5 as Moderate priority.

Create an Excel workbook called Skills_Gap_Analysis.xlsx in your workspace with four sheets. The first sheet should be named Curriculum_Coverage and contain columns for course identifier, course name, enrollment count, assignment count, quiz count, and total assessments. Include one row per course. The second sheet should be named Workforce_Profile and contain columns for department name, headcount, average education level description, and average performance rating. Include one row per department. The third sheet should be named Gap_Matrix and contain columns for skill area, curriculum score, industry demand score, gap score, and priority level. Include one row for each of the four skill areas. The fourth sheet should be named Recommendations and contain columns for priority, gap area, recommended action, and implementation timeline. Include at least four actionable recommendations based on the gap analysis findings.

Build a presentation called Skills_Gap_Presentation.pptx in your workspace with at least five slides. The first slide should be a title slide with the presentation name and date. Include slides covering the curriculum analysis findings, the workforce profile summary, the gap matrix results, and the recommendations. Each content slide should have a clear title and substantive bullet points or data summaries.

Finally, schedule three advisory board meeting events on the shared calendar for the week of March 16 to March 20, 2026. The first meeting on Monday at 10 AM for two hours should focus on curriculum review. The second meeting on Wednesday at 2 PM for ninety minutes should cover workforce data findings. The third meeting on Friday at 9 AM for two hours should be for gap analysis recommendations. Each event summary should include "Advisory Board" and the meeting topic.
