You are an academic advisor preparing end-of-semester reviews for the Global Governance and Geopolitics program. Your goal is to analyze student performance data from two course sections, identify students who need academic advising, publish analytics, and set up the advising infrastructure.

Start by reading the Advising_Handbook.pdf in your workspace, which describes the GPA thresholds for different advising categories and the procedures for scheduling advising appointments. Also read the semester_config.json file, which specifies the courses to analyze and the grading scale parameters.

From the learning management system, retrieve data for two courses: the Global Governance and Geopolitics course from Fall 2013 with identifier 20 and the same course from Fall 2014 with identifier 21. For each course, get the course name, total enrolled students, the number of assignments, and the number of quizzes. Also retrieve all student submissions for these courses, including scores for each assignment. The submission scores are needed to compute grade distributions.

Using command-line tools, write and execute a Python script called advising_analyzer.py in your workspace. The script should compute the following analytics for each course: the average submission score, the median submission score, and the pass rate defined as the percentage of submissions scoring 60 or above out of 100. Create a grade distribution by grouping scores into ranges: below 60 as failing, 60 to 69 as D range, 70 to 79 as C range, 80 to 89 as B range, and 90 to 100 as A range. Count the number of submissions in each range and calculate percentages. Based on the overall analysis, categorize advising needs: students in the failing range need Urgent Intervention, those in the D range need Academic Support, those in the C range need Progress Monitoring, and those in B or A range need no action. Estimate the number of students needing each type of advising based on the submission score distributions.

Create an Excel workbook called Academic_Advising_Report.xlsx in your workspace with four sheets. The first sheet named Course_Summary should contain columns for course identifier, course name, enrollment count, average score, median score, and pass rate percentage. Include one row per course. The second sheet named Grade_Distribution should contain columns for grade range label, count of submissions in that range, and percentage of total submissions. Include one row for each of the five grade ranges across both courses combined. The third sheet named Advising_Needs should contain columns for advising category, estimated student count, and the required action description. Include one row for each category: Urgent Intervention, Academic Support, Progress Monitoring, and No Action Required. The fourth sheet named Appointment_Schedule should contain columns for date, time, and the number of available advising slots. Include entries for five weekdays in the week of March 16 to March 20, 2026, with three slots per day at 9 AM, 11 AM, and 2 PM.

Publish the grade analytics to a cloud spreadsheet. Create a spreadsheet titled "Academic Advising Analytics" with at least one sheet containing the course summary data including course names, enrollment numbers, average scores, and pass rates.

Create a Word document called Advising_Recommendations.docx in your workspace. The document should include a title referencing academic advising for the Global Governance program, an overview section summarizing the performance data across both course sections, a section identifying at-risk students based on the failing and D-range submission counts, and a recommendations section with specific actions for each advising category.

Create a knowledge base database called "Student Advising Tracker" with properties for Student Category as a title field, Count as a number field, Action Required as a rich text field, and Priority as a select field with options High, Medium, and Low. Add one page entry for each advising category with the appropriate data.

Schedule advising appointment blocks on the shared calendar for the week of March 16 to March 20, 2026. Create five events, one per weekday, each from 9 AM to 4 PM. The event summary should include "Academic Advising" and the day of the week. Add a description mentioning Global Governance program advising sessions.
