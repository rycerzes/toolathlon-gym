The HR leadership team wants a comprehensive employee engagement analysis that combines data warehouse insights with new survey instruments and a knowledge base entry. There is a PDF called HR_Engagement_Policy.pdf in the workspace that outlines the company's engagement strategy and target scores by department. There is also a JSON file called engagement_targets.json with numeric targets for each metric. Please read both files first to understand the policy context and targets.

Start by querying the employee data warehouse to get department-level statistics. For each of the 7 departments, calculate the average job satisfaction score, average performance rating, average work-life balance score, and the total employee count. Round all averages to 2 decimal places and sort alphabetically by department.

Next, compute an engagement index for each department. The engagement index is calculated as the weighted average of the three metrics: job satisfaction contributes 40 percent of the weight, work-life balance contributes 35 percent, and performance rating contributes 25 percent. Round this index to 2 decimal places.

Using the command line, write and run a Python script that reads in these department metrics and computes additional analytics. The script should determine which department has the highest engagement index, which has the lowest, the overall company average engagement index, and classify each department as either Above Average or Below Average relative to the company mean. Save the script output to a file called engagement_analysis_output.txt in the workspace.

Create a survey form titled "Employee Engagement Survey" with exactly 5 questions. Question 1 asks how satisfied the respondent is with their current role on a scale of 1 to 10. Question 2 asks about work-life balance satisfaction with options Very Satisfied, Satisfied, Neutral, Dissatisfied, and Very Dissatisfied. Question 3 asks about career growth opportunities with options Excellent, Good, Fair, and Poor. Question 4 asks how supportive their direct manager is using a 1 to 5 scale. Question 5 asks whether they would recommend the company as a great place to work with options Yes, Maybe, and No.

Create a page in the knowledge base titled "HR Engagement Dashboard" that contains a summary of the department statistics including the department names, employee counts, and engagement indices.

Create an Excel file called Employee_Engagement_Report.xlsx in the workspace with three sheets. The first sheet should be called "Department_Scores" with columns Department, Avg_Satisfaction, Avg_Performance, Avg_Work_Life_Balance, Employee_Count, and Engagement_Index. Each row represents one department sorted alphabetically. The second sheet should be called "Survey_Design" with columns Question_Number, Question_Text, and Response_Type listing the 5 survey questions. The third sheet should be called "Action_Items" with columns Priority, Department, and Recommendation. Include one action item row for each department, where the priority is High for departments below the company average engagement index and Medium for those above. The recommendation should mention the specific area where that department scores lowest among the three metrics.

Finally, send an email to hr-leadership@company.com with the subject "Employee Engagement Analysis Report" and a body that summarizes the key findings including which department scored highest and lowest on the engagement index.
