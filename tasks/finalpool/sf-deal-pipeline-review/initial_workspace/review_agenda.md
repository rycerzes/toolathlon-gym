# Quarterly Pipeline Review Meeting Agenda

## Meeting Structure

1. Welcome and objectives (5 minutes)
2. Regional performance overview (15 minutes)
3. Segment-level deep dive (20 minutes)
4. Root cause analysis for underperforming areas (15 minutes)
5. Action items and next steps (10 minutes)
6. Q&A (10 minutes)

## Key Discussion Points

- Compare actual Q1 2025 performance against pipeline targets
- Identify regions and segments that are behind target
- Discuss corrective actions for underperforming areas
- Set revised targets if necessary

## Attendees

- VP Sales (chair)
- Regional Sales Directors
- Segment Leaders
- Finance Business Partner
- Sales Operations

## Pre-Meeting Preparation

- Review the Pipeline_Review.xlsx report
- Prepare explanations for any significant gaps
- Come with proposed corrective actions
