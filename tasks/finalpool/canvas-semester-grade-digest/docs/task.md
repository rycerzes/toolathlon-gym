The academic affairs office needs a comprehensive grade summary for all courses offered in the Fall 2014 semester. Please connect to the learning management system, retrieve the list of courses from that term, and for each course calculate the enrollment count along with the average, minimum, and maximum student scores.

Create an Excel file called Grade_Digest.xlsx in the workspace with two sheets. The first sheet should be named "Course Grades" and contain columns Course_Name (the full course name including the semester designation), Enrolled_Students, Avg_Score (rounded to two decimal places), Min_Score (rounded to two decimal places), and Max_Score (rounded to two decimal places). Sort rows alphabetically by course name. Only include students who have a score recorded.

The second sheet should be named "Summary" with columns Metric and Value. Include these entries: Total_Courses (number of Fall 2014 courses), Total_Students (total enrolled students with scores across all courses), Overall_Average (weighted average score across all courses rounded to two decimal places, weighted by enrollment), Highest_Avg_Course (full name of the course with the highest average score), Lowest_Avg_Course (full name of the course with the lowest average score), and Courses_Below_70 (the count of courses where the average score is strictly below 70).

Additionally, create a Word document called Grade_Summary.docx in the workspace that contains a brief narrative paragraph summarizing the overall findings including the total number of courses, total students, overall average, and which course performed best and worst.
