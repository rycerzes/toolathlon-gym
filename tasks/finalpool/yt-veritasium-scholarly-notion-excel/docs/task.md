You are an education researcher building a science resource library that connects popular science videos with relevant academic papers. Your task is to identify the top Veritasium science videos by viewership and find related academic papers for each one.

Start by querying the video database to retrieve all videos from the Veritasium channel. Select the 5 most-viewed videos. For each of the 5 videos, identify the main scientific topic based on the video title. For example, a video about zooming in microscopy would have the topic "microscopy" or "fractals", and a video about a skyscraper engineering problem would have the topic "structural engineering".

For each of the 5 identified topics, search the academic paper database for relevant papers. Collect up to 3 papers per topic. Record the paper title, authors, publication year, and citation count for each paper found.

Create a spreadsheet file called Science_Resource_Map.xlsx in your workspace. This file must contain two sheets.

The first sheet must be named Videos. It should have one row per top video with the following columns in order: Rank (1 through 5, ranked by view count descending), Video_ID, Title, View_Count, Main_Topic, and Paper_Count (the number of academic papers you found for that topic). The rows must be sorted by Rank ascending.

The second sheet must be named Papers. It should have one row per paper found across all topics. The columns must be in this order: Topic, Paper_Title, Authors, Year, and Citations. If no papers are found for a topic, that topic may be omitted from this sheet.

After creating the spreadsheet, create a page in the team knowledge base titled exactly "Science Video-Paper Resource Map". The page content should be written as flowing text that lists the 5 Veritasium videos with their topics and the number of related papers found for each. The text should be formatted as readable paragraphs, not as a table or list with special characters.
