Science Education Resource Map Project

Goal: Connect top Veritasium science videos with relevant academic papers to create a curated learning resource.

Process:
1. Retrieve top 5 Veritasium videos by view count from the video database
2. Identify the main scientific topic for each video based on its title
3. Search the academic paper database for papers related to each topic (up to 3 papers per topic)
4. Create Science_Resource_Map.xlsx with Videos and Papers sheets
5. Create a team knowledge base page titled "Science Video-Paper Resource Map"

Output files:
- Science_Resource_Map.xlsx
  - Videos sheet: Rank, Video_ID, Title, View_Count, Main_Topic, Paper_Count
  - Papers sheet: Topic, Paper_Title, Authors, Year, Citations
- Team knowledge base page: "Science Video-Paper Resource Map"
